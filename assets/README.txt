ASSETS FOLDER — drop your files here
=====================================

The website looks for these files by name. Just copy your files into this
folder and rename them to match exactly. No code changes needed.

1) PROFILE PHOTO
   File name:  firat-barca.jpg
   Where it shows: the round avatar at the top of the homepage.
   Tips: square image works best (e.g. 600x600 px). .jpg or rename a .png to .jpg.
   If this file is missing, the site shows your initials "FB" instead.

2) CV / RESUME (PDF)
   File name:  Firat-Barca-CV.pdf
   Where it shows: the "Download CV" button in the homepage hero.
   If this file is missing, the button will give a 404 until you add it.

3) (Optional) FAVICON — the little icon in the browser tab
   Ask Claude to wire this up if you want one.

HOW TO ADD A FILE
-----------------
- In Finder, drag the file into this "assets" folder.
- Rename it to the exact name above (capitalisation matters).
- Refresh the website in your browser.

Want different file names or more slots? Just ask.
